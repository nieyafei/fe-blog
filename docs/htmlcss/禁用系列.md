# 禁用系列

## 禁用鼠标事件

```css
.disabled {
    pointer-events: none;
    cursor: default;
    opacity: 0.6;
}
```

## 禁止用户选择

```css
body{
  -webkit-touch-callout: none;
  -webkit-user-select: none;
  -khtml-user-select: none;
  -moz-user-select: none;
  -ms-user-select: none;
  user-select: none;
}
```
